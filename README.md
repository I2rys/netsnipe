# NetSnipe
Simple Discord bot that snipes Discord gifts.

## Installation
Github:
```
git clone https://github.com/I2rys/netsnipe
```

NpmJS:
```
npm i node-json-db randomstring discord.js@13.16.0 request-async lodash fs discord.js-selfbot-v13
```

## Usage
```
node index.js
```
